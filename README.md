# Intoduction

- This project is a web application that helps farmers to get information about crops, diseases, and pests.
- This project leverages the power of AI to provide farmers with accurate and up-to-date information.

---

# Features

- Crop information
- Disease information
- Pest information
- Weather information
- News information
- Agmarknet information
- YouTube information
- Groq Chatbot information
